# Haijiao Lab 网站

这是一个参考 Wei Lab 信息架构制作的中文课题组静态网站初版，已替换为 Haijiao Lab、毛海蛟主任医师和宁波大学附属第一医院相关信息。

## 预览

打开 index.html 即可在浏览器中预览。网站不需要构建步骤、依赖安装或后端服务。

## 替换内容

- 替换页眉、首页和页脚中的实验室名称、联系方式和地址。
- 替换实验室简介中的真实研究定位和学科背景。
- 替换研究方向为团队真实课题。
- 替换团队成员信息，包括姓名、职称、研究方向和个人照片。
- 替换论文成果为真实论文题目、作者、期刊、年份和链接。
- 替换科研资源为真实临床队列、样本库、数据库、协议或技术平台。
- 替换新闻动态为团队真实动态。
- 替换加入我们中的岗位、申请条件和联系邮箱。

## 文件

- `index.html`：页面结构和主要中文内容。
- `styles.css`：响应式视觉样式和 logo 展示样式。
- `script.js`：平滑滚动、移动端导航、栏目高亮和论文筛选。
- `assets/hailiao-lab-logo.png`：Haijiao Lab Logo。

正式发布前，请确认邮箱、地址、论文、团队成员、科研项目和伦理/合规信息均为真实可公开内容。
