# Haijiao Lab 官网

这是 Haijiao Lab 的课题组官网，依托宁波大学附属第一医院建设，集中展示团队简介、研究方向、核心成员、代表性论文、科研资源和新闻动态。

## 网站内容

- 实验室简介：介绍 Haijiao Lab 的临床背景、研究定位和团队建设方向。
- 研究方向：围绕跟腱损伤、组织修复、疾病机制、精准诊疗和转化医学展开。
- 团队成员：展示毛海蛟、姚立炜、毛旭锋、张壮伟等核心成员信息。
- 论文成果：按年份整理团队代表性论文，并标注毛海蛟的作者署名。
- 科研资源：说明临床队列、样本资源和技术平台的支撑基础。
- 新闻动态：记录团队相关报道和阶段性进展。
- 加入我们：提供合作交流和人才联系信息。

## 文件结构

- `index.html`：网站页面结构和中文内容。
- `styles.css`：页面视觉样式、响应式布局和图片展示规则。
- `script.js`：导航交互、栏目高亮和论文年份筛选。
- `assets/`：网站使用的 logo、favicon、成员照片和科研图片。
